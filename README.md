# Smart Mess Management and Shared Expense Tracking System

A web-based project for managing bachelor/mess life, including meal tracking, shared expenses, rent collection, utility bills, and monthly summaries.

## Project Idea

Many students and working people in Bangladesh live together in flats or messes. In this lifestyle, meal counting, bazar cost, rent payment, and utility bills are usually managed manually. This creates confusion, mistakes, and lack of transparency.

This project aims to solve that problem through a centralized web application.

## Main Features

- User authentication
- Mess/member management
- Daily meal entry
- Expense tracking
- Rent management
- Utility bill tracking
- Due and balance calculation
- Monthly report generation
- Notifications and reminders

## Technology Stack

- **Frontend:** React.js
- **Backend:** Spring Boot
- **Database:** MySQL (XAMPP)
